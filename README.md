### Kurulum ve Çalıştırma

```bash
pnpm install
pnpm dev
```

Alternatif olarak:

- npm: `npm install && npm run dev`
- yarn: `yarn && yarn dev`

### Scriptler

- `pnpm dev`: Geliştirme sunucusu (Turbopack)
- `pnpm build`: Production build
- `pnpm start`: Production sunucusu
- `pnpm lint`: ESLint kontrolü
